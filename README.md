# 📊 Sentiment Analysis using Python

## 📌 Project Overview
This project performs sentiment analysis on textual data using Natural Language Processing (NLP). It classifies text into Positive, Negative, and Neutral sentiments.

## 🎯 Objective
The objective of this project is to analyze customer reviews and automatically identify their sentiment to help businesses understand customer opinions.

## ⚙️ How it Works
1. Load dataset using Pandas  
2. Apply TextBlob for sentiment analysis  
3. Classify each sentence as Positive, Negative, or Neutral  
4. Store results in a new file  
5. Visualize sentiment distribution using charts  
6. Generate WordCloud for negative reviews  

## 📊 Results
- Positive reviews: Majority  
- Negative reviews: Moderate  
- Neutral reviews: Few
